# Tử Vi Quick Reference - Star Combinations Cheat Sheet

> **Trường phái:** Tam Hợp Phái (cổ truyền VN) + Bảng Tứ Hóa Trung Châu Phái (Vương Đình Chi, biến thể VN)

## 🔍 CÁCH ĐỌC NHANH

### Nhìn cung nào trước?

1. **Mệnh** → Tổng thể con người
2. **Tật Ách** → Sức khoẻ (đặc biệt kiểm tra hung tinh)
3. **Phúc Đức** → Tâm linh, âm phần, mồ mả
4. **Điền Trạch** → Nhà cửa, bất động sản
5. **Phu Thê** → Hôn nhân
6. **Tài Bạch** → Tiền tài
7. **Quan Lộc** → Sự nghiệp

### Xung chiếu tính nhanh

```
Cung vị trí X → Đối cung = (X + 6) % 12
Tam hợp = (X + 4) % 12 và (X + 8) % 12
Giáp = (X + 1) % 12 và (X - 1 + 12) % 12
```

## ⚡ BỘ SAO CẦN QUÉT NGAY

### Khi thấy ở Phúc Đức

| Bộ sao | Nghĩa | Action |
|--------|--------|--------|
| Tang Môn + Điếu Khách | Mồ mả bất ổn | → RS03 (fire!) |
| Cô Thần + Quả Tú | Cô đơn tâm linh | → RS05 |
| Hỷ Thần + Thiên Phúc + Tấu Thư | Lập/sửa bàn thờ | → RS04 |
| Thiên Khốc + Thiên Hư | Âm khí nặng | → RS06 |
| Liêm-Hình-Phá | Mộ bị phá | → RS06 (critical) |

### Khi thấy ở Tật Ách

| Bộ sao | Nghĩa | Action |
|--------|--------|--------|
| Kình + Đà (song chiếu) | Tai nạn | → H01 |
| Thiên Hình + Bạch Hổ | Mổ xẻ | → H02 |
| Liêm + Sát | Bệnh máu | → H03 |
| Thái Dương hãm + Kỵ | Mắt kém | → H04 |
| Hoả + Linh (nhẹ) | Va chạm | → H05 |

### Khi thấy ở Phu Thê

| Bộ sao | Nghĩa | Action |
|--------|--------|--------|
| Hồng Loan + Thiên Hỷ | Cưới hỏi | → C01 |
| Đào Hoa + Hoá Kỵ | Ngoại tình / rắc rối | → RC06 |

### Khi thấy ở Quan Lộc

| Bộ sao | Nghĩa | Action |
|--------|--------|--------|
| Cự Môn + Hoá Kỵ | Kiện tụng | → RC01 |
| Tử Vi + Hoá Quyền | Thăng tiến | → C04 |

### Khi thấy ở Tài Bạch

| Bộ sao | Nghĩa | Action |
|--------|--------|--------|
| Không + Kiếp | Phá sản | → Pattern #4 |
| Thiên Mã + Lộc Tồn + Hoá Lộc | Đại phát | → C05 |

## 🌟 14 CHÍNH TINH - Miếu/Hãm Nhanh

```
Tử Vi:     Ngọ(6)▲ Tỵ(5)▲ — Không hãm
Thiên Cơ:  Tý(0)▲ Dần(2)▲ — Tỵ(5)▼ Dậu(9)▼
Thái Dương: Mão(3)▲ Thìn(4)▲ Tỵ(5)▲ Ngọ(6)▲ — Dậu(9)▼ Tuất(10)▼ Hợi(11)▼ Tý(0)▼
Vũ Khúc:   Thìn(4)▲ Tuất(10)▲ — Mão(3)▼ Dậu(9)▼
Thiên Đồng: Dần(2)▲ Thân(8)▲ — Tỵ(5)▼ Hợi(11)▼
Liêm Trinh: Thân(8)▲ Dậu(9)▲ — Tỵ(5)▼ Hợi(11)▼
Thiên Phủ:  Sửu(1)▲ Mùi(7)▲ Tý(0)▲ Ngọ(6)▲ — Mão(3)▼ Dậu(9)▼
Thái Âm:   Dậu(9)▲ Tuất(10)▲ Hợi(11)▲ Tý(0)▲ — Mão(3)▼ Thìn(4)▼ Tỵ(5)▼ Ngọ(6)▼
Tham Lang:  Thìn(4)▲ Tuất(10)▲ — Tỵ(5)▼ Hợi(11)▼
Cự Môn:    Tý(0)▲ Thìn(4)▲ — Tỵ(5)▼ Hợi(11)▼
Thiên Tướng: Sửu(1)▲ Mùi(7)▲ — Tỵ(5)▼ Hợi(11)▼
Thiên Lương: Tý(0)▲ Ngọ(6)▲ — Tỵ(5)▼ Hợi(11)▼
Thất Sát:  Dần(2)▲ Thân(8)▲ — Thìn(4)▼ Tuất(10)▼
Phá Quân:  Tý(0)▲ Ngọ(6)▲ — Thìn(4)▼ Tuất(10)▼
```

▲ = Miếu/Vượng (tốt)  |  ▼ = Hãm (xấu)

## 🔁 LƯU TỨ HOÁ - 10 CAN

| Can | Hoá Lộc | Hoá Quyền | Hoá Khoa | Hoá Kỵ |
|-----|---------|-----------|----------|--------|
| Giáp(0) | Liêm Trinh | Phá Quân | Vũ Khúc | Thái Dương |
| Ất(1) | Thiên Cơ | Thiên Lương | Tử Vi | Thái Âm |
| Bính(2) | Thiên Đồng | Thiên Cơ | Văn Xương | Liêm Trinh |
| Đinh(3) | Thái Âm | Thiên Đồng | Thiên Cơ | Cự Môn |
| Mậu(4) | Tham Lang | Thái Âm | Hữu Bật | Thiên Cơ |
| Kỷ(5) | Vũ Khúc | Tham Lang | Thiên Lương | Văn Khúc |
| Canh(6) | Thái Dương | Vũ Khúc | Thái Âm | Thiên Đồng |
| Tân(7) | Cự Môn | Thái Dương | Văn Khúc | Văn Xương |
| Nhâm(8) | Thiên Lương | Tử Vi | Tả Phụ | Vũ Khúc |
| Quý(9) | Phá Quân | Cự Môn | Thái Âm | Tham Lang |

**Song Kỵ nguy hiểm:**

- Năm sinh Can X → Hoá Kỵ Gốc = sao A
- Năm xem Can Y → Lưu Hoá Kỵ = sao B
- Nếu A và B cùng cung → **Song Kỵ** (cực hung)
- Nếu A ở Đại Vận + B chiếu → **Kỵ trùng phùng** (rất xấu)
